# stock_workspace

AI 기반 주식 분석 프로젝트. 이번 라운드는 빈 골격만 둔다 — 상세 설계(데이터 수집, 분석 방법, 매매 불변조건 등)는 다음 단계에서 별도로 진행한다.

`claude_codex_workspace`와 마찬가지로 레포 루트의 [`../chat_rooms/`](../chat_rooms/), [`../runtime_logs/`](../runtime_logs/)를 공용으로 사용할 수 있다.

## 현재 상태

설계 전. `claude_codex_workspace`의 Room/Task 운영 모델이 검증되면 동일 패턴을 이 워크스페이스에도 확장 적용할 예정([`../docs/STANDARD_RULES.md`](../docs/STANDARD_RULES.md) 참조).
